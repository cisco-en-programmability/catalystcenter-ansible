Index of all Collection Environment Variables

The following index documents all environment variables declared by plugins in collections. Environment variables used by the ansible-core configuration are documented in Ansible Configuration Settings.

CATALYSTCENTER_DEBUG

Enable SDK debug logging.

Used by: cisco.catalystcenter.catalystcenter inventory plugin

CATALYSTCENTER_HOST

Hostname or IP address of the Catalyst Center appliance.

Used by: cisco.catalystcenter.catalystcenter inventory plugin

CATALYSTCENTER_PASSWORD

Password for Catalyst Center authentication.

Used by: cisco.catalystcenter.catalystcenter inventory plugin

CATALYSTCENTER_PORT

TCP port for the Catalyst Center API.

Used by: cisco.catalystcenter.catalystcenter inventory plugin

CATALYSTCENTER_USERNAME

Username for Catalyst Center authentication.

Used by: cisco.catalystcenter.catalystcenter inventory plugin

CATALYSTCENTER_VERIFY

Whether to validate SSL certificates.

Used by: cisco.catalystcenter.catalystcenter inventory plugin

CATALYSTCENTER_VERSION

Catalyst Center API version string.

Used by: cisco.catalystcenter.catalystcenter inventory plugin
